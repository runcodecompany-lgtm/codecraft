1. Guest Access System

Implement a complete Guest vs Authenticated User experience.

Guests must be able to:

- View Landing Page
- View Courses
- View Learning Tracks
- View Teachers
- View Community Discussions
- View Questions & Answers
- View Public Profiles
- Search Content

Guests must NOT be able to:

- Enroll in Courses
- Join Learning Tracks
- Participate in Community
- Create Posts
- Create Questions
- Create Replies
- Create Comments
- React to Content
- Access Student Dashboard
- Access Teacher Dashboard
- Access Private Features
- Download Certificates

When a guest attempts any restricted action:

Show login/signup flow.

Never allow unauthorized actions.
